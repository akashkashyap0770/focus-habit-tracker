## Project Overview
Focus & Habit Tracker is a full-stack web application to help users:
- Track daily activities/habits
- View historical logs
- Analyze weekly productivity with animated charts (Three.js)

---

## Features
- Authentication: Signup/Login with JWT
- Daily Log: Add/Delete activities with name, duration, category, timestamp
- History: Calendar/date-grouped view
- Analytics: Weekly overview with bar charts (Three.js)
- UI/UX: Smooth transitions, hover effects, responsive design

---

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Authentication: JWT
- Visualization: Three.js
- Deployment: Netlify (frontend) + Render (backend)
